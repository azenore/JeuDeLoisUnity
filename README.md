# JeuDeLoisUnity

Dans ce jeu 1 objectif avoir assez d'argent pour payer ses taxes

Il y a des cases malus, bonus, dialogues, neutre et mini jeu

Le mini jeu est simple il faut clicker sur les carrés de couleurs pour fuir les deux policiers du labyrinthe pendant 30sec



A terme j'aimerais que ce jeu ai plus de mécaniques autours des lois et des règles et plus de mini jeux décallés.



Pour faire avancer le pion il suffit de "roll the dice"
